# Candle Labeler

An interactive **human-in-the-loop** labeling tool for training a multi-timeframe trading model on candlestick charts.

You label price action across 6 different lookback windows (300, 200, 100, 50, 20, 10 candles), and an online-learning neural network learns from your decisions in real time.

---

## Features

- **Interactive Dash + Plotly** candlestick charts with volume profile
- Click anywhere on chart to draw horizontal reference lines
- 6 simultaneous browser windows (one per timeframe)
- Real-time online learning model (PyTorch)
- Persistent replay buffer + running normalization
- Auto-saves model and labels after every batch

## Installation

Clone the repository:
```Bash
git clone https://github.com/yourusername/candle-labeler.git
cd candle-labeler
```
Create a virtual environment (recommended):
```Bash
python -m venv venv
source venv/bin/activate          # On Windows: venv\Scripts\activate
```
Install dependencies:
```Bash
pip install -r requirements.txt
```

## Setup (Run Once)

1. Place your raw tick/second-level data in the `data/` folder.
Example filename: `xauusd_s1_20260529_20260628.csv`
2. Run the setup script:
   ```Bash
   python setup_data.py
   ```

This will:

- Convert raw data → 1-minute candles (`data/xauusd_1min.csv`)
- Create `my_data.csv` with correct headers


## Usage
```Bash
python main.py
```

## Workflow

- A random candle is selected
- 6 browser tabs open automatically (one per timeframe)
- Label each chart as Buy, Neutral, or Sell
- After all 6 are labeled → data is saved and model trains
- Press Enter for next batch, or type `0` to quit


## Model

- Multi-head neural network (shared backbone + 6 heads)
- Online learning with replay buffer
- Running feature normalization per timeframe
- Model auto-saves to `model_dir/online_model.pth`


## Notes & Warnings

- `main.py` kills other Python processes at the start of each batch to clean up old ports.
*Do not run this on a machine with other important Python scripts running.*
- The app opens 6 browser tabs simultaneously.
- Model improves over time as you label more data.


*Made for traders who want to systematically teach their edge to a model.*
